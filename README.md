I am a Cloud Reliability Engineer with a passion for empowering developers
through automation. My free time is spent building web apps with Deno and
Vue.js, deployed to Google Cloud Run, all managed through
infrastructure-as-code.

## Repositories

### [Infrastructure](https://github.com/dmikalova/infrastructure)

Terramate stacks with OpenTofu for managing personal infrastructure. Includes
GCP resources, GitHub repository configuration, and Supabase databases. Secrets
encrypted with SOPS and Age.

### [Email Unsubscribe](https://github.com/dmikalova/email-unsubscribe)

Automated email unsubscribe system that connects to Gmail via OAuth and
processes unsubscribe requests. Uses Playwright for browser automation to handle
complex unsubscribe flows. Built with Deno, Hono, Vue.js, and PostgreSQL.

### [Login](https://github.com/dmikalova/login)

Centralized authentication service for mklv.tech apps using Supabase Auth. Built
with Deno and Hono.

### [github-meta](https://github.com/dmikalova/github-meta)

Shared configurations and reusable workflows for all repositories. Provides base
lefthook pre-commit hooks and GitHub Actions workflows.

### [Factorio Mods](https://github.com/dmikalova/factorio-mods)

Collection of small Factorio mods available on
[the Factorio mod portal](https://mods.factorio.com/user/dmikalova).

### [Dotfiles](https://github.com/dmikalova/dotfiles)

Personal configuration files managed with [Chezmoi](https://www.chezmoi.io/).

### [Brocket window manager](https://github.com/dmikalova/brocket)

A run-or-raise window manager for Linux written in Go. Useful for keyboard
shortcuts to open, switch to, or resize windows.
